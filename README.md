# Excel-Sales-Dashboard
Sales Performance Dashboard created using Excel with Pivot Tables and Slicers

## Project Overview
This project analyzes sales data using Microsoft Excel.

## Features
- Revenue, Profit, and Margin Analysis
- Pivot Tables
- Interactive Slicers
- Region-wise Performance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Visualization

## Key Insights
- Identified top performing regions
- Analyzed product-wise revenue
- Compared salesperson performance
